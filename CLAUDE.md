# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 저장소 현재 상태

이 저장소는 초기화 단계이며 다음 파일만 존재합니다.

- `welcome.txt` — `Hello, Terminal` 한 줄만 담긴 텍스트 파일

아직 소스 코드, 빌드 설정(package.json, pyproject.toml 등), 테스트, README, git 이력이 없습니다. 빌드 · 린트 · 테스트 명령어, 아키텍처, 디렉터리 구조에 대해 기록할 사실이 없으므로 이 문서에는 추정으로 채우지 않습니다.

## 향후 작업을 위한 안내

- 새로운 코드를 추가할 때는 사용자에게 **언어 · 스택 · 실행 방식**을 먼저 확인하세요. 임의로 프레임워크나 도구 체인을 선택하지 마세요.
- 실제 코드 · 설정 파일이 생기면 이 문서를 다음 항목으로 갱신하세요.
  - 자주 쓰는 명령어 (빌드, 실행, 단일 테스트 실행 방법 포함)
  - 여러 파일을 읽어야 파악되는 큰 그림의 아키텍처
- 사용자 요청은 한국어로 들어올 수 있습니다. 사용자가 별도로 지시하지 않는 한 한국어로 응답하세요.
